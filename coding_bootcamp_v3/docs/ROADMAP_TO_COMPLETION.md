# Roadmap to Completion After v28

## Current: v28 - One Task at a Time Bootcamp

The app now uses a gated course flow:

```text
Lesson + Coaching -> Questions -> Testing + Verification -> Saved Review -> Next Lesson
```

## v29 - Adaptive Remediation

When a learner misses a question or fails verification, the app should route them to a targeted mini-lesson instead of simply saying wrong.

## v30 - More Real Exercises Per Topic

Each lesson should have multiple verified exercises at beginner, intermediate, and professional difficulty.

## v31 - Interview Mode

Add strict one-question-at-a-time interview drills based on the Google-style coaching flow.

## v32 - Final Course Polish

Improve copy, spacing, command output clarity, and package quality.

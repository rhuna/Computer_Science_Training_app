# 12-Month CS Bootcamp Plan

## Month 1: Foundations
Binary, files, memory, runtimes, terminal basics, Git basics, and build/run workflow.

## Month 2: Programming Core
Variables, functions, collections, error handling, debugging, and small CLI apps in at least two languages.

## Month 3: Data Structures
Arrays, maps, stacks, queues, linked lists, trees, graphs, and Big-O.

## Month 4: Algorithms
Sorting, searching, recursion, BFS, DFS, two pointers, sliding windows, hash map patterns, and interview drills.

## Month 5: Systems
Processes, threads, filesystems, networking, HTTP, JSON, and OS fundamentals.

## Month 6: Databases
SQL, relational modeling, keys, indexes, transactions, migrations, and reporting.

## Month 7: Web and APIs
Frontend state, backend APIs, auth, validation, full-stack structure, and deployment planning.

## Month 8: Software Engineering
Testing, architecture, refactoring, documentation, design patterns, release notes, and maintainability.

## Month 9: Security and Reliability
Threat modeling, secure coding, logs, backups, incident response, and operational readiness.

## Month 10: Cloud and DevOps
Docker concepts, CI/CD, environments, build scripts, release checklists, and observability.

## Month 11: Specialization
Choose one path: full-stack, backend, systems/desktop, AI/data, game/simulation, mobile, security, or DevOps.

## Month 12: Capstone and Career
Finish a serious capstone, polish the README, add screenshots, prepare demo talking points, write resume bullets, and practice interviews.

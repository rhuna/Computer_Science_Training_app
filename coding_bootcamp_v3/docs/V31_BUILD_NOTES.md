# v31 Build Notes

## Intent

v31 is a learning-experience upgrade. It does not reintroduce clutter or many tabs. It keeps the two-side-panel / one-center-task layout from v29-v30, but makes the center task more course-like.

## Key changes

- Renamed the version to CS Bootcamp Desktop v31.
- Updated the runtime database to `progress_v31.sqlite`.
- Added scrollable task stages.
- Added a dedicated Knowledge Check stage before coding.
- Improved the stylesheet for a less mechanical interface.
- Expanded lesson text generation to include stronger explanation and example sections.
- Updated the gated stage order to Lesson, Coaching, Knowledge Check, Testing, Verification, Save.

## Build

Run:

```bat
rebuild_clean.bat
```

# v31 - SoloLearn-Style Course Experience

v31 focuses on making the app easier and more pleasant to learn from, not just adding more topics.

## Main goal

The app now behaves more like a guided interactive course:

1. Lesson
2. Coaching
3. Knowledge Check
4. Testing
5. Verification
6. Save and move to the next lesson

Only the current step is shown in the center workspace. This keeps the learner from being overwhelmed while preserving the professional functionality behind the scenes.

## Lesson improvements

Lessons now include:

- A lesson goal
- Why the topic matters
- A core explanation
- A concrete example section
- A worked thinking template
- Step-by-step completion instructions
- Checklist guidance with mini example prompts
- Practice tasks broken into small actions

The lesson area is scrollable so longer explanations do not crowd the rest of the app.

## Coaching improvements

The coaching phase is meant to help the learner move toward the solution instead of jumping straight to code. It asks the learner to:

- Restate the prompt
- Define input and output
- Choose a problem-solving pattern
- Trace a tiny example
- Implement the smallest correct version
- Verify behavior
- Debug the first wrong state
- Refactor
- Explain correctness and complexity

## Knowledge check

v31 adds a dedicated Knowledge Check step between coaching and testing. This makes the test feel connected to the lesson. The learner answers a checkpoint question and reads the explanation before coding.

## Testing and verification

Testing remains separated from verification:

- Testing is where the learner writes, builds, runs, and experiments.
- Verification is where the app runs real correctness checks against expected behavior.

This preserves the v23+ real verification model while making it easier for the learner to understand when they are practicing versus when they are proving correctness.

## UI polish

v31 adds:

- Softer colors
- Larger cards
- Better spacing
- More readable lesson text
- Scrollable stage panels
- Clearer left and right side panels
- Less mechanical step flow

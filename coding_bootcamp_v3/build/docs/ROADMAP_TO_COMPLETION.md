# Roadmap to Completion After v31

## Current: v31 - Complete CS Coverage Curriculum

v31 keeps the clean step-by-step interface and expands the topic catalog so the learner becomes well-rounded instead of only practicing isolated coding problems.

The app now covers:

- Programming fundamentals
- Data structures and algorithms
- Mathematical foundations
- Computer architecture and operating systems
- Networking and internet systems
- Databases and storage
- Software engineering quality
- HCI, accessibility, and product thinking
- Security, privacy, ethics, and responsible engineering
- Cloud, DevOps, deployment, and observability
- Advanced systems, compilers, distributed systems, and AI/RAG architecture

## v31 - Adaptive Remediation

When a learner fails a quiz or verifier, the app should route them to a smaller prerequisite lesson and targeted practice.

## v35 - Deeper Real Test Harness

Expand the verifier with per-exercise structured test specs, stdin/stdout tests, file-output tests, and language-native unit test execution.

## v35 - Guided Interview Mode

Add one-question-at-a-time interviewer mode for the selected lesson, with scoring, hints, and saved feedback.

## v35 - Curriculum Completion Analytics

Add a mastery map showing which CS coverage areas are complete, weak, in progress, or ready for capstone work.

## v31 completed

v31 upgraded the learning experience from a mechanical workflow into a cleaner guided course experience with extensive explanations, examples, coaching, knowledge checks, testing, verification, and saved review.

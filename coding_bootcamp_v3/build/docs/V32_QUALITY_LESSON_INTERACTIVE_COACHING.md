# v35 - Quality Lesson and Interactive Coaching Pass

v35 is a content-quality rebuild. It replaces generic filler-style lesson text with topic-specific lessons that explain the actual concept, why it matters, a worked example, and a guided practice path.

## New course flow

The app keeps the focused step-by-step interface:

1. Lesson
2. Coaching
3. Knowledge Check
4. Testing
5. Verification
6. Save

## Lesson quality standard

Every generated module now includes lessons with:

- Topic-specific explanation
- Concrete worked example
- Lesson objective
- Checklist with how-to learning actions
- Practice tasks tied to the topic
- Preview of the coaching problem

## Coaching quality standard

Coaching is now tied to the selected lesson and includes:

- A coached problem
- Guided interviewer-style questions
- Fill-in-the-blank planning prompts
- Dropdown-style decision prompts
- Code-completion scaffold
- A writable coaching answer box

The intent is to help the learner solve the problem rather than simply tell them to solve it.

## Testing quality standard

Knowledge checks and tests are attached to the same module the learner just studied. This makes the testing portion reinforce the lesson instead of feeling random.

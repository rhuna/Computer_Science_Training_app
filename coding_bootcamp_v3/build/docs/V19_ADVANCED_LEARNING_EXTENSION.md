# v19 - Advanced Learning Extension

v19 extends CS Bootcamp Desktop beyond job-readiness into deeper, senior-level learning.

## Purpose

The earlier versions build a professional foundation: CS fundamentals, language tracks, exercises, algorithm visualization, project generation, portfolio building, Git/CI/CD, career readiness, SQLite tracking, instructor mode, AI tutoring scaffolding, model adapters, release polish, and final capstone certification.

v19 adds a deeper extension for learners who want to push into advanced engineering topics.

## Advanced domains

- Advanced algorithms
- Systems and OS internals
- Database internals
- Distributed systems
- Compilers and language implementation
- Performance engineering
- Advanced architecture
- Advanced security engineering
- AI/data/RAG systems
- Engineering leadership artifacts

## Completion expectation

A learner should produce advanced artifacts, not just read lessons:

- Implementations
- Tests
- Benchmarks
- Design docs
- Diagrams
- Runbooks
- Security reviews
- Failure-mode reviews
- Portfolio-ready READMEs

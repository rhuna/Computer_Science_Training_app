# v9 Git, Testing, CI/CD, and Deployment Lab

v9 adds the professional workflow layer that connects CS learning to real software engineering practice.

## New Page

The new **Git Testing CI/CD Lab** page teaches and drills:

1. Git fundamentals and commit hygiene.
2. Branching, pull requests, and code review.
3. Unit testing and test-driven debugging.
4. GitHub Actions CI pipelines.
5. Docker and local service packaging.
6. Release checklists and deployment readiness.

## Why This Version Matters

A developer is not job-ready just because they can write code. Professional software work also requires:

- clean Git history,
- branches,
- tests,
- automated validation,
- reproducible setup,
- release notes,
- deployment discipline,
- documentation that another developer can follow.

v9 starts turning every generated project and portfolio project into a professional workflow exercise.

## Export

The new lab exports:

```txt
engineering_exports/v9_engineering_readiness_checklist.md
```

That checklist can be copied into project repositories as a quality gate before publishing a portfolio project.

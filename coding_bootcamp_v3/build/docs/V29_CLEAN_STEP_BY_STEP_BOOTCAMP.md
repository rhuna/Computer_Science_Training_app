# CS Bootcamp Desktop v31 - Complete CS Coverage Curriculum

## Goal

v31 removes the clutter from the guided course interface. Instead of showing lesson text, coaching, testing, verification, notes, and history all in one crowded view, the center panel now shows only one task at a time.

## Flow

1. **Lesson** - read the objective, explanation, checklist, and practice goal.
2. **Coaching** - use the timer and helper prompts to plan the solution before coding.
3. **Testing** - write code, build, run, and submit the attempt.
4. **Verification** - run real correctness tests. Fake PASS markers are not accepted as proof.
5. **Save** - write a reflection, save the finished lesson, and move to the next lesson.

## Side Panels

- The **left panel** shows available lessons only.
- The **right panel** shows finished/saved lessons only.

This keeps the user oriented without overloading the center workspace.

# CS Bootcamp Desktop v10 - Career Readiness Center

v10 adds the final learner-facing career layer before database, instructor, AI tutor, and release polish versions.

## New Page

**Career Readiness Center**

The page helps the learner move from "I completed lessons" to "I can present myself professionally for a junior role."

## Features

- Target role selector.
- Role-specific weekly plan.
- Interview drill generator.
- Behavioral interview practice prompts.
- Technical interview practice prompts.
- System design practice prompts.
- Role readiness scorecard.
- Resume bullet builder.
- Markdown export to `career_exports/v10_career_readiness_plan.md`.

## Target Roles

- Junior Software Engineer.
- Backend/API Engineer.
- Full-Stack Developer.
- C++ Desktop / Systems Developer.
- Python Automation / Data Engineer.
- Cloud / DevOps Associate.
- Security-Minded Software Engineer.
- Game / Simulation Developer.

## Professional Standard

A learner should be able to:

1. Explain their target role.
2. Present 3 polished projects.
3. Discuss testing, Git, and CI/CD basics.
4. Explain one API and one database design.
5. Walk through debugging in a calm professional way.
6. Convert project work into resume bullets.
7. Practice behavioral and technical answers out loud.

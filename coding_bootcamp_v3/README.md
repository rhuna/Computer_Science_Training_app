# CS Bootcamp Desktop v35 - Split-Topic Deep Lessons

v35 is a curriculum-quality rebuild. Combined lessons have been split into individual focused lessons so the learner studies one concept at a time. For example, hash maps, sets, and unordered sets are now separate lessons with separate coaching problems, tests, and verification targets.

## Core flow

Lesson -> Coaching -> Testing -> Verification -> Save -> Next Lesson

## v35 focus

- One objective per lesson
- No bundled multi-topic lessons
- Detailed topic-specific explanations
- Clear coding problem for every lesson
- Solution walkthrough tied to that exact problem
- Testing question tied to the same objective
- Separate exercise metadata for each topic
- Cleaner curriculum progression from beginner fundamentals to advanced systems

rebuild_clean.bat

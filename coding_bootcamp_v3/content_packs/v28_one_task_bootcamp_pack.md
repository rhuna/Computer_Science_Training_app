# V28 One Task Bootcamp Pack

This content pack supports the v28 gated course flow.

## Course rhythm

1. Read the lesson.
2. Use the coach to restate the problem.
3. Answer the checkpoint question.
4. Write the smallest correct implementation.
5. Run real verification.
6. Save review notes.
7. Move to the next lesson.

## Review questions saved after each task

- What was the problem asking for?
- What inputs and outputs mattered?
- What pattern did I choose?
- What failed first?
- What fixed the failure?
- What is the time complexity?
- What would I improve next?

# v31 Interactive Course Experience Pack

Use every lesson in this order:

1. Read the lesson goal.
2. Read why the topic matters.
3. Study the example.
4. Complete the checklist mini examples.
5. Use coaching before coding.
6. Answer the knowledge check.
7. Code the smallest correct solution.
8. Run tests.
9. Verify correctness.
10. Save the reflection.

The learner should not be shown all tools at once. The app should reveal the next task only when it is useful.

# v24 Problem Solving Efficiency Pack

Use this pack as the default practice rhythm for every lesson.

## Daily session

1. Pick one module and one exercise.
2. Read the lesson objective.
3. Open Problem Solving Coach.
4. Choose a pattern.
5. Complete one 20-minute drill.
6. Verify the code.
7. Review one quiz question.
8. Record the result in progress.

## Weekly targets

- 5 verified exercises
- 5 written explanations
- 3 failed-test debugging reviews
- 2 complexity comparisons
- 1 algorithm visualization walkthrough

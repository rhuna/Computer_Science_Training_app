# v27 Learning / Testing Course Content Pack

This pack explains the new course direction.

## Instructor style

The app should teach like a Google-style interviewer:

- Ask one question at a time.
- Make the learner explain before coding.
- Push on complexity, edge cases, and correctness.
- Make verification part of the workflow.
- Keep the learner from wandering through unrelated tabs.

## Course screen sections

1. Course module and language selection.
2. Lesson explanation.
3. Advanced helper checkpoint.
4. Real-time problem-solving timer.
5. Connected exercise.
6. Code editor.
7. Build/run/verify controls.
8. Quiz checkpoint.
9. Algorithm mental model.
10. Progress evidence.

# v35 Quality Lessons Pack

This pack defines the new lesson quality rules used in CS Bootcamp Desktop v35.

Each lesson should answer:

1. What is this topic?
2. Why does it matter professionally?
3. What is a concrete example?
4. What pattern or structure does the learner need?
5. What can go wrong?
6. How does the learner prove they understand it?

Each coaching section should include:

- Guided questions
- Fill-in-the-blank prompts
- Dropdown-style decisions
- Code-completion scaffold
- Verification strategy

The course should behave less like a static textbook and more like an interactive bootcamp instructor.

# Prompt Safety Checklist

Before sending a prompt to a hosted or local model, confirm:

- [ ] No API keys or passwords are included.
- [ ] No resident/patient/private health information is included.
- [ ] No private company source code is included unless approved.
- [ ] The learner understands what will be sent.
- [ ] The output will be reviewed instead of blindly copied.
- [ ] The final learning artifact includes tests, notes, or a commit.

# v25 Advanced Helper Content Pack

## Phase prompts

1. Rewrite the problem in your own words.
2. Write input, output, constraints, and edge cases.
3. Choose a pattern and justify it.
4. Trace the smallest useful example.
5. Implement the smallest correct version.
6. Run real verification.
7. Debug the first wrong state.
8. Refactor and rerun verification.
9. Explain correctness, complexity, and tradeoffs.

## Use this every day

Run one 20-minute helper session per exercise. The learner should improve speed by repeating the same professional sequence until it becomes automatic.

# v24 Problem Solving Efficiency Course

v24 improves the app for learners who need to become faster, more accurate coding problem solvers.

## Main goal

The app now teaches a repeatable process instead of only presenting lessons and exercises.

The course loop is:

1. Learn the concept.
2. Classify the problem.
3. Choose a pattern.
4. Model input and output.
5. Hand-trace a small example.
6. Implement the simplest correct solution.
7. Verify with real tests.
8. Debug the first wrong state.
9. Refactor.
10. Explain correctness and complexity.

## New tab

`Problem Solving Coach` connects the selected module, lesson, exercise, and language track. It gives pattern-specific guidance and a 20-minute deliberate practice drill.

## Patterns included

- Universal problem-solving loop
- Input/output modeling
- Brute force to optimized
- Two pointers
- Hash map lookup
- Stack/queue state
- Tree/graph traversal
- Dynamic programming
- Debugging failed tests
- Complexity and tradeoff review

## Why this matters

Most learners do not fail because they cannot type code. They fail because they skip understanding, examples, constraints, tracing, and verification. v24 turns those steps into a visible workflow.

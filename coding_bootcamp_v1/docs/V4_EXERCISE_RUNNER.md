# v4 Exercise Runner

v4 turns the CS Bootcamp Desktop app from a read-and-track curriculum into a hands-on practice environment.

## What the runner does

The new **Exercise Runner** page lets the learner:

- Select an industry language track.
- See the starter folder for that language.
- Open or browse to the exercise folder.
- Run a default command.
- Build or compile the starter.
- Run a basic test/check command.
- View command output inside the app.
- Stop a running process.

## Safety model

The runner is intentionally local and explicit. It does not download or execute unknown code by itself. It runs commands only against the selected folder and is intended for the starter exercises included in this repo or folders the learner personally created.

## Supported starter tracks

- C++
- Python
- JavaScript
- TypeScript
- Java
- C#
- Go
- Rust
- SQL
- Kotlin

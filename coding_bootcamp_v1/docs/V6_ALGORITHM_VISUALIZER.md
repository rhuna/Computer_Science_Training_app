# v6 Algorithm Trainer and Visualizer

v6 adds the first algorithm visualization layer to the CS Bootcamp Desktop app.

## Included Visualizations

- Bubble Sort
- Selection Sort
- Binary Search
- Stack push/pop
- Queue enqueue/dequeue
- BFS graph traversal
- DFS graph traversal
- Hash table lookup

## Learning Goal

The visualizer is designed to help a learner understand what an algorithm is doing before writing code. The correct workflow is:

1. Step through the algorithm visually.
2. Explain the algorithm in plain English.
3. Write the algorithm in one language.
4. Write tests for edge cases.
5. Rebuild the same algorithm in another language.
6. Document Big-O time and space complexity.

## Why This Matters

Professional developers do not memorize algorithms only for interviews. They use algorithmic thinking when designing APIs, caches, queues, schedulers, indexes, tree walkers, graph traversal tools, search systems, data pipelines, and game/simulation logic.

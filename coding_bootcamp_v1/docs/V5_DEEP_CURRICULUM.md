# v5 Deep CS Curriculum Expansion

v5 expands the project from a multi-language trainer into a true computer science bootcamp platform.

## New v5 Modules

### 07 - Web Engineering
Covers frontend architecture, UI state, backend APIs, authentication, authorization, validation, and service boundaries.

### 08 - Data, AI, Automation, and RAG Engineering
Covers data pipelines, CSV cleaning, RAG concepts, document Q&A architecture, evaluation sets, and AI product guardrails.

### 09 - Secure Coding and Reliability
Covers threat modeling, input validation, secrets handling, backups, logging, incident notes, and safe failure design.

### 10 - Cloud, DevOps, CI/CD, and Deployment
Covers Git workflows, release checklists, Docker concepts, environment variables, pipeline stages, and deployment discipline.

### 11 - Capstone, Portfolio, Interview, and Career Readiness
Covers portfolio quality, resume bullets, coding interview patterns, beginner system design, and final capstone presentation.

## Professional Goal

The goal is to move the learner from:

1. knowing syntax,
2. to solving problems,
3. to building complete apps,
4. to understanding systems,
5. to shipping professional portfolio projects.

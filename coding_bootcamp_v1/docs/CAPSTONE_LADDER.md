# Capstone Ladder

The bootcamp should produce several levels of portfolio evidence.

## Level 1: Starter Utilities
- Base converter
- File byte inspector
- Word frequency counter
- Input validator

## Level 2: Data-backed Tools
- Learning tracker database
- SQL reporting project
- Notes app with persistence
- CSV analyzer

## Level 3: Full Applications
- Full-stack learning tracker
- Qt desktop dashboard
- REST API with database
- Algorithm visualizer

## Level 4: Professional Capstones
- Document Q&A assistant
- Secure notes system
- Deployable API service
- Mini SaaS dashboard
- World builder or simulation tool

## Level 5: Job-Ready Presentation
Every final capstone should include:

- Problem statement
- User story
- Screenshots or demo video
- Architecture diagram
- Build/run instructions
- Tests or test plan
- Security/reliability notes
- Future improvements
- Resume bullets

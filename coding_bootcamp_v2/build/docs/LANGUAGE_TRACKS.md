# Language Tracks

v2 supports these industry language tracks:

- C++: systems, desktop, game engines, high-performance apps
- Python: automation, AI, data, scripting, backend APIs
- JavaScript: frontend web, Node.js, browser apps
- TypeScript: professional frontend/backend web apps
- Java: enterprise backend, JVM systems, Android foundations
- C#: Windows apps, .NET backend, Unity games
- Go: cloud services, networking, DevOps tools
- Rust: safe systems programming, CLI tools, infrastructure
- SQL: databases, analytics, reports, backend data access
- Kotlin: Android, JVM apps, modern mobile development

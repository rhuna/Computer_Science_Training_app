# V27 Learning / Testing Tabs Content Pack

## Recommended learner flow

1. Open Learning + Coaching.
2. Pick a module and language.
3. Read the lesson.
4. Use the helper pop-up to answer the first checkpoint.
5. Trace a tiny example.
6. Switch to Testing + Verification.
7. Code the exercise.
8. Build, run, and verify with real tests.
9. Take the quiz checkpoint.
10. Record the drill.

## Instructor framing

The learner should not rush straight into code. The app now makes that easier by keeping all understanding work in one tab and all proof/testing work in another tab.

# v27 Learning / Testing Course Mode

v27 simplifies the app into one focused bootcamp workspace.

## Why

The previous app had many useful systems, but too many tabs made the learner think about navigation instead of problem solving. v27 keeps the systems but presents them in one guided flow.

## Learner workflow

1. Choose module and language.
2. Read the selected lesson.
3. Use the helper pop-up as the current instructor question.
4. Work through the real-time 20-minute problem-solving timer.
5. Select the connected exercise.
6. Write code in the same screen.
7. Build, run, and verify with real test cases.
8. Answer the connected quiz question.
9. Step through the related algorithm visualizer.
10. Record progress evidence.

## Design rule

One screen. One current module. One current lesson. One current exercise. One current quiz. One selected language.

The app should feel like a course, not a pile of tools.

# v23 Guided Course Bootcamp

v23 is a course-first restructuring build.

## Removed tabs

The app no longer exposes AI Tutor Mode, Model Adapter Lab, Notes, Project Generator, or Advanced Learning Extension as separate tabs. The goal is to keep the learner focused on a clear bootcamp path.

## Integrated advanced learning

Advanced learning now appears inside the core course flow:

- Advanced lessons are inside the Learning Path.
- Advanced exercises are inside the Practice Lab.
- Advanced verification is handled by the Exercise Runner.
- Advanced quiz questions are inside the Quiz Trainer.
- Advanced algorithms and systems traces are inside Algorithm Visualizer.

## Course flow

Module -> Lesson -> Practice -> Runner -> Verify -> Quiz -> Progress.

## Advanced topics included

- Distributed systems
- Database internals
- B-tree indexing
- Compilers and tokenizers
- Performance engineering
- Security architecture
- Token bucket rate limiting
- AI/RAG retrieval pipelines

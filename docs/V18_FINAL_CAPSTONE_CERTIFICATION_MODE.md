# v18 - Final Capstone and Certification Mode

v18 completes the bootcamp flow by adding a final capstone and certification workflow.

## Main goals

- Choose a role-aligned final capstone.
- Build a complete professional project with README, tests, screenshots, release notes, and architecture notes.
- Prepare a mock interview packet explaining design decisions, tradeoffs, bugs, testing, and improvements.
- Export a completion report that can be reviewed by an instructor, mentor, or future employer.

## Evidence required

- Product spec
- Architecture notes
- Build/run/test commands
- Demo screenshots or walkthrough notes
- Automated tests or manual QA checklist
- Release notes
- Portfolio README
- Mock interview packet
- Instructor or mentor sign-off
